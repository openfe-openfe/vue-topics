# vue-topics

## 这是什么(What is it)
vue仿github标签功能

## 代码演示如何使用

npm install vue-topics --save 

1.导入并注册组件,注册组件
```js
import {Topics} from 'vue-topics'
components: {
   Topics
}
```
2.template模板
```html
   <topics/>
```

## 组件演示demo

```js
git clone https://github.com/openfe-openfe/vue-topics.git
cd vue-topics
npm install 
npm run dev
```
